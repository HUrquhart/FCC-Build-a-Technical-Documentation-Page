Working with container elements

<div class="container">

Containers help items stay aligned on the site and help keep the content in the center of the screen when the screen size is larger then the container width

Containers use maxwidth and margin combined with a small amount of padding

  • Add containers as a class
  • Containers are better suited a children of other elements so that they may provide backgrounds when needed

Buttons

<a class="btn primary">Button text</a">

When using buttons you can apply a combination of 2 clases btn and any status helper class to style the element

You customise the variables in the SCSS partials to change the colour scheme see below for a list of variables used

  • $primary-color
  • $secondary-color
  • $tertiary-color
  • $color-4
  • $color-5
